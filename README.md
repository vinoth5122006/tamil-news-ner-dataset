# Tamil News NER & Headline Generation Dataset
This repository contains a Tamil news dataset designed for **Named Entity Recognition (NER)** and **automatic headline generation**.
---
## Objective
To build a system that:
* Extracts named entities from Tamil news text
* Identifies key elements like **Person, Location, Organization, Date, etc.**
* Generates **concise and meaningful headlines** using extracted entities
---
## Problem Statement
Tamil news articles are often lengthy and contain rich information.
Users need **quick, clear, and informative headlines** to understand the news efficiently.
---
## Proposed Solution
This project focuses on:
* Processing Tamil news text
* Extracting important entities using NER
* Using extracted entities to generate **crispy headlines**
---
##  Dataset
The dataset includes:
* Tamil news content
* Structured format (CSV / Excel)
* Suitable for NLP tasks like NER and summarization
---
## Features
* Tamil text processing
* Named Entity Recognition (NER) ready dataset
* Supports headline generation tasks
* Useful for NLP research and applications
---
## Use Cases
* News headline generation
* Tamil NLP research
* Information extraction
* Text summarization systems
